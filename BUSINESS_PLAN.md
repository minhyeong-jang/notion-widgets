## 실행 로드맵

### Phase 0: 제품화 (8주) — 수익 0원
- 쿼리 파라미터 시스템 + Notion embed 최적화
- D-Day 카운트다운 + 명언 위젯
- 랜딩 페이지 (갤러리 + 커스터마이저 + URL 생성기)
- 테마 시스템 3종 + 워터마크
- 서비스 이름 + 도메인 + GA4 + SEO

### Phase 1: 첫 수익 (2-4개월) — 목표 월 10만원
- 한국 Notion 커뮤니티 포스팅
- Ko-fi / GitHub Sponsors 설정
- Product Hunt 런칭
- Build in Public (Twitter/X)

### Phase 2: 안정 수익 (4-8개월) — 목표 월 50-100만원
- Lemon Squeezy Pro 플랜 런칭
- 영어 UI 추가
- Vercel 이전 (API 서버)
- 위젯 8-10종 확보
- 사업자등록 (간이과세자)

### Phase 3: 확장 (8-18개월) — 목표 월 300만원+
- 구독 플랜 추가
- 크리에이터 제휴 프로그램
- Notion 마켓플레이스 입점
- Obsidian/Craft 확장
