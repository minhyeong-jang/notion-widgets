# Notion 위젯 서비스 경쟁사 분석 & 차별화 전략

## 1. 주요 경쟁사 분석

### 1-1. Indify (indify.co)

**포지셔닝:** Notion 위젯 시장 1위. Notion, Canva, Obsidian, Coda, Google Sites 등 크로스 플랫폼 지원.

**제공 위젯 (10종):**
- Google Calendar / Outlook Calendar (프리미엄)
- Weather (프리미엄)
- Image Gallery (프리미엄)
- Life Progress Bar
- Countdown Timer
- Quotes (인스타그램 계정 연동)
- Counter (+ - / ▲ ▼)
- Clock (다양한 스타일: Digital Roulette, Analog Smooth, Analog Dot 등)

**수익모델:**
- Free: 최대 6개 위젯, 기본 커스터마이징
- Pro (위젯별 과금): Google Calendar, Weather 등 고급 위젯 개별 유료 (~$8/mo)
- Creator (계정 단위): 무제한 위젯 + 대시보드 + Pro 업그레이드 1개 포함

**추가 수익원:**
- 프리메이드 대시보드 판매 (여행 플래너 등 테마별 대시보드)
- Notion 템플릿 크리에이터와의 협업 (위젯 포함 템플릿)

**강점:** Product Hunt / Indie Hackers 출신, 커뮤니티 기반 성장, 크로스 플랫폼 확장
**약점:** 위젯 수 자체는 10종으로 많지 않음, 디자인 커스터마이징이 제한적

---

### 1-2. WidgetBox (widgetbox.app)

**포지셔닝:** Notion 전용 위젯 플랫폼, 심미성(Aesthetic) 강조. 50만 개 이상의 위젯 생성 실적.

**제공 위젯 (13종+):**
- Digital Retro Clock (플립 시계) / Analog Clock
- Calendar
- Weather (Round / Square / 5-day Forecast)
- Like Button / Page View Tracker / Upvote Button
- Life Progress Bar
- Quote Widget
- Countdown Timer
- Simple Button (URL 리다이렉트)
- AI Widgets (신규)

**수익모델:**
- Free: 5개 위젯 (캘린더, 시계, 명언만), 전체 커스터마이징 가능
- Pro: $1.99/월 — 전체 위젯 + 무제한 생성

**강점:** 저렴한 가격, 인터랙티브 위젯 (좋아요, 조회수 등), AI 위젯 도입
**약점:** Notion에만 한정, 위젯별 깊이가 얕음

---

### 1-3. Apption (apption.co)

**포지셔닝:** Notion 위젯 마켓플레이스/디렉토리. 직접 만들기보다 큐레이션 중심.

**제공 위젯 (100+종, 외부 연동 포함):**
- 기본: Countdown, Clock, Calendar, Weather, Quotes
- 유니크: 달력(음력), 환율, Google Analytics, 뉴스레터 구독, 화이트보드
- 엠비언스: 캠프사이트/빈티지 테마 ASMR + 로파이 플레이리스트
- 유틸: 스톱워치, 계산기, 마인드맵, 주간 스케줄러
- 외부 임베드 변환: HTML → Notion embed URL 컨버터

**수익모델:**
- 대부분 무료, 광고 기반 또는 외부 서비스 연결 수수료로 추정
- 직접적인 구독 모델은 없음

**강점:** 위젯 수 압도적, 커뮤니티 기여 기반, 다양한 니치 위젯
**약점:** 품질 편차 큼, 통일된 UX 없음, 일부 위젯 셋업 복잡

## 2. 시장 공통 수익모델 패턴

### Freemium (가장 보편적)
- **무료**: 기본 위젯 3~6개, 제한된 커스터마이징
- **유료**: $2~8/월, 전체 위젯 + 무제한 생성 + 고급 커스터마이징

### 위젯별 과금
- Indify 방식: 특정 위젯만 개별 Pro 업그레이드

### 번들/대시보드 판매
- 위젯 + Notion 템플릿을 묶어서 일회성 판매 ($5~30)

### 크리에이터 생태계
- 템플릿 크리에이터가 위젯을 포함한 템플릿 판매 → 수수료/제휴

## 3. 경쟁사들의 공통 약점 (우리의 기회)

### 3-1. 한국/아시아 시장 부재
- 전 서비스가 영어 기반, 한국어 UI 없음
- 한국 Notion 사용자는 매우 많지만 한국 타겟 위젯 서비스는 없음
- 음력 달력, 한국 공휴일, D-Day(수능/입대 등) 등 로컬 니즈 미충족

### 3-2. 계정 생성 강제
- Indify, WidgetBox 모두 위젯 생성 전 회원가입 필수
- "빠르게 URL 하나 복사해서 Notion에 붙여넣기" 경험이 아님

### 3-3. 디자인 차별화 부족
- 대부분 비슷한 플랫 디자인, 위젯 간 시각적 통일성 없음
- 우리 프로젝트의 플립 시계처럼 디테일 있는 애니메이션은 드뭄

### 3-4. 모바일 최적화 부족
- Notion 모바일에서 위젯이 깨지거나 비율이 안 맞는 경우 많음

### 3-5. Notion 외 플랫폼 확장
- Obsidian, Craft, Bear 등 다른 노트앱 사용자는 선택지가 거의 없음

## 4. 차별화 전략 제안

### 전략 1: "No Sign-up, Just Copy" — 제로 프릭션 경험
- **회원가입 없이** URL 쿼리 파라미터만으로 커스터마이징
- 예: `yoursite.com/widget/clock?color=ff6b6b&format=24h&theme=flip`
- 랜딩 페이지에서 실시간 미리보기 + URL 자동 생성
- 경쟁사 대비 가장 빠른 위젯 적용 경험 → 바이럴 포인트

### 전략 2: "Premium Aesthetic" — 디자인 퀄리티로 승부
- 현재 플립 시계의 애니메이션 품질을 전 위젯에 적용
- 테마 시스템: Minimal / Retro / Glassmorphism / Neubrutalism
- 경쟁사가 제공하지 않는 수준의 마이크로 인터랙션
- Notion 위젯계의 "Apple"을 지향

### 전략 3: 한국 시장 선점 + 글로벌 확장
- 1차: 한국어 UI + 한국 특화 위젯 (음력, D-Day, 공휴일 카운트다운)
- Notion 한국 커뮤니티 (네이버 카페, 디스코드) 공략
- 2차: 영어 UI 추가 → 글로벌 확장

### 전략 4: 수익모델 설계

**Phase 1 - 무료로 사용자 확보**
- 전체 위젯 무료 제공 (커스터마이징 포함)
- 위젯 하단에 작은 "Powered by ○○" 워터마크

**Phase 2 - 프리미엄 전환**
- 워터마크 제거: $1.99/월 또는 $15/년
- 프리미엄 테마 팩: $3~5 일회성
- 프리미엄 위젯 (날씨, 캘린더 연동 등): 개별 과금

**Phase 3 - 크리에이터 생태계**
- Notion 템플릿 크리에이터에게 위젯 번들 제공
- 위젯 포함 템플릿 판매 시 수수료 수익

## 5. 우선순위 위젯 로드맵 (제안)

### MVP (지금 → 2개월)
1. ✅ 플립 시계 (이미 완성) — 쿼리 파라미터 커스터마이징 추가
2. ✅ Life Progress Bar (이미 완성) — 목표일/생년월일 커스터마이징 추가
3. 🆕 카운트다운 타이머 (D-Day) — 한국 시장 핵심 위젯
4. 🆕 명언 위젯 — 한/영 명언 지원
5. 🆕 랜딩 페이지 — 위젯 미리보기 + URL 생성기

### Growth (2~4개월)
6. 포모도로 타이머
7. 아날로그 시계 (다양한 스타일)
8. 캘린더 위젯
9. 날씨 위젯
10. 테마 시스템 (최소 3종)

### Scale (4~6개월)
11. Google Calendar 연동
12. 습관 트래커
13. 프리미엄 구독 시스템
14. 위젯 빌더 (사용자가 직접 위젯 커스텀)

## 6. 핵심 경쟁 우위 요약

```
경쟁사: 회원가입 → 위젯 선택 → 커스터마이징 → 링크 복사 → Notion에 붙여넣기
우리:   랜딩 페이지에서 커스터마이징 → URL 복사 → Notion에 붙여넣기 (끝)
```

- **속도**: 가입 없이 30초 만에 위젯 적용
- **디자인**: 경쟁사 대비 압도적인 애니메이션/비주얼 퀄리티
- **로컬**: 한국 시장 특화 (경쟁사 공백 지대)
- **가격**: 기본 무료, 프리미엄은 업계 최저 수준
